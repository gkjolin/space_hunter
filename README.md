# Space Hunter

Overview :
- I am preparing a 2D flight shooting game demo project using the recently updated vertical scrolling background graphic. I captured the existing 3D units to 2D textures and configured the project only with 2D artwork to reduce the capacity and improve the performance. There are related demo images and demo game links, so please refer to those who are interested and please give me a message if you are graphic artist who will design the flight units for a fee. :) 
- 최근에 업데이트 했던 종스크롤 배경 그래픽을 활용해서 2D 비행 슈팅 게임 데모 프로젝트를 준비하고 있는데 기존 3D 유닛들을 2D로 캡쳐해서 2D 아트웍으로만 프로젝트를 구성해서 용량도 줄이고 성능 개선도 시켜봤습니다. 관련 데모 영상과 데모 게임 링크가 있으니 관심있는 분들은 참고해주시고 혹시나 출연하는 비행 유닛들을 유료로 디자인해주실 그래픽 아티스트가 있으시면 메시지 주세요. :)
